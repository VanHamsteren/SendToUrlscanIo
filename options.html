Extension options

This extension allows you to submit URLs for assessment using urlscan.io. To use this feature, you need to register for an account and obtain an API key.

  • To register an account, visit: urlscan.io Signup
  • To obtain an API key, visit: urlscan.io API Key

Once you obtained your API key, please fill out the details below. Keep your API-key private and never share it, because the key is personal and bound to your account. This extension only stores the API-key in localy, using Firefox storage database.
